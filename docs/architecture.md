# Portfolio Architecture

## Overview
A high-performance, premium developer portfolio built for the LOrdEnRYQuE brand. Designed with a "Dark Luxury" aesthetic featuring "Muted Metallic Gold" accents and sophisticated motion design.

## Tech Stack
- **Framework**: Next.js 16.1.6 (App Router + Turbopack)
- **Styling**: Tailwind CSS v4 (Custom design tokens)
- **Animations**: Framer Motion (Liquid Motion, staggered reveals, pulsing indicators)
- **Internationalization**: Custom `I18nProvider` (EN/DE support)
- **Icons**: Lucide React (Refined, consistent iconography)
- **UI Components**: Shadcn-inspired custom glass cards and metallic buttons

## Key Components
- **Navbar**: Sticky glassmorphism with dynamic mobile menu.
- **Hero**: Liquid motion background with editorial-style typography.
- **DemosSection**: Interactive walkthroughs for specific niches (SaaS, Restaurant, Beauty).
- **Client Portal**: Simulation-mode demonstration of project management transparency.
- **AI Concierge**: Custom chat-based navigation and support portal.

## Design Principles
1. **Premium First**: No generic colors; use bespoke HSL-tailored metallic tints.
2. **Motion as Feedback**: Subtle hover states and pulsing indicators for "live" system feel.
3. **Typographic Hierarchy**: High-contrast editorial headers vs. technical mono-spaced accents.
